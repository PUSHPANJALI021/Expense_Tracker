# 💰 Expense Tracker

A simple and efficient **Expense Tracker web application** built using **Vite + JavaScript** to help users track income and expenses easily.

---

## ✨ Features

* ➕ Add income and expenses
* 🧾 View transaction history
* 📊 Real-time balance update
* 🗑️ Delete transactions
* 💡 Simple and responsive UI
* ⚡ Fast performance with Vite

---

## 🛠️ Tech Stack

* ⚛️ Frontend: HTML, CSS, JavaScript
* ⚡ Build Tool: Vite
* 🎨 Styling: CSS
* 📦 Package Manager: npm

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/PUSHPANJALI021/Expense_Tracker.git
```

### 2. Navigate to project folder

```bash
cd Expense_Tracker
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run development server

```bash
npm run dev
```

### 5. Open in browser

```
http://localhost:5173
```

---

## 📦 Build for Production

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```
