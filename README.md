Expense Tracker

A simple and efficient Expense Tracker web application built using Vite + JavaScript to help users track income and expenses easily.

 Features
➕ Add income and expenses
🧾 View transaction history
📊 Real-time balance update
🗑️ Delete transactions
💡 Simple and responsive UI
⚡ Fast performance with Vite
🛠️ Tech Stack
⚛️ Frontend: HTML, CSS, JavaScript
⚡ Build Tool: Vite
🎨 Styling: CSS
📦 Package Manager: npm
📁 Project Structure
Expense_Tracker/

⚙️ Installation & Setup
1. Clone the repository
git clone https://github.com/PUSHPANJALI021/Expense_Tracker.git
2. Navigate to project folder
cd Expense_Tracker
3. Install dependencies
npm install
4. Run development server
npm run dev
5. Open in browser
http://localhost:5173
📦 Build for Production
npm run build

To preview build:

npm run preview
